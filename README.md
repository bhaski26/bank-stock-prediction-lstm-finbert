# Bank Stock Price Prediction using LSTM + FinBERT

## Overview
This project predicts stock prices of major Indian banks using:
- LSTM Deep Learning
- FinBERT Sentiment Analysis
- Yahoo Finance Live Data

## Features
- Multi-day stock prediction
- Real-time market data integration
- Sentiment-enhanced forecasting
- Streamlit interactive dashboard
- Buy/Hold/Sell recommendation

## Tech Stack
- Python
- TensorFlow / Keras
- Scikit-learn
- Streamlit
- yFinance

## Run Project

```bash
pip install -r requirements.txt
streamlit run gui.py
